== Web Fonts

.Font erst lokal suchen, bevor die URL geladen wird
[source, css, role="code"]
----
@font-face
{
font-family: 'Mplus1';
font-weight: 100;
src: local('Mplus1-Thin'),
     url('Mplus1-Thin.woff2') format('woff2'),
     url('Mplus1-Thin.woff') format('woff');
}
----
